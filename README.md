**VPN panel 5X-UEV** — это ультра-современная, легкая и быстрая веб-панель управления для мультипротокольного VPN/Proxy сервера с поддержкой VLESS (Reality, xhttp), Hysteria 2, NaiveProxy и Mita (Mieru). Панель создана с упором на максимальную простоту использования, высокую производительность и управление в режиме одного окна. Забудьте о сотнях перегруженных параметров: добавляйте домены и создавайте клиентов за пару секунд!

---

## ✨ Ключевые преимущества

*   **Режим одного окна**: Управление статусом системных служб, клиентами, доменами и ядрами протоколов собрано на одном лаконичном экране.
*   **Мультипротокольность**: Поддержка пяти самых стойких к блокировкам протоколов:
    *   `VLESS + Reality` (по умолчанию, не требует домена, работает по IP-адресу).
    *   `VLESS + xhttp` (высокая имитация веб-трафика по протоколу HTTP/2 через WebSocket).
    *   `NaiveProxy` (работает внутри веб-сервера Caddy, идеальная маскировка под обычный HTTPS-сайт).
    *   `Hysteria 2` (протокол на базе UDP/QUIC, гарантирует высокую скорость на нестабильных каналах связи).
    *   `Mita (Mieru)` (TCP мультиплексирование для скрытия сигнатуры прокси).
*   **Автоматический выпуск SSL-сертификатов**: Панель сама запрашивает бесплатные сертификаты Let's Encrypt через Certbot при привязке домена.
*   **Интерактивный помощник (Wizard)**: Встроенный пошаговый гид проведет вас от настройки DNS-записей до добавления первого пользователя и копирования готовой конфигурации.
*   **Встроенные QR-коды**: Генерируйте QR-коды для быстрого импорта конфигурационных ссылок в мобильные приложения.
*   **Встроенный BBR & Sysctl**: Автоматическая оптимизация сетевых настроек ядра Linux и BBR для снижения пинга и ускорения передачи данных.
*   **Элегантный темный дизайн**: Премиальный веб-интерфейс с использованием стеклянного эффекта (glassmorphism), плавных переходов и интуитивно понятных элементов управления.

## 💻 Системные требования

*   **ОС**: Ubuntu 20.04 / 22.04 / 24.04 (рекомендуется чистая система)
*   **Архитектура**: x86_64 (AMD64)
*   **Права**: Доступ к пользователю `root` (или sudo)
*   **Сеть**: Открытые порты 80 и 443. Внешний IPv4-адрес.

## 🛠️ Установка в одну команду

Для автоматической установки всех зависимостей, настройки веб-сервера Nginx, сборки Caddy с NaiveProxy, установки Xray Core, Mieru, Hysteria 2 и развертывания веб-панели запустите следующую команду в терминале вашего сервера:

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/pasnya/vpn-panel-5x-uev/main/install.sh)"
```

### После завершения установки вы получите:
1.  **Reality Public Key** для ручной настройки.
2.  **Секретный URL** для входа в веб-панель управления (например, `http://IP_СЕРВЕРА:ПОРТ/СЕКРЕТНЫЙ_ПУТЬ/index.php`).
3.  **Логин и пароль администратора** по умолчанию:
    *   *Логин:* `admin`
    *   *Пароль:* `admin_password_change_me` (*обязательно измените пароль в базе данных при первом входе!*)

## 📖 Быстрый старт (Интерактивный мастер)

После входа в веб-интерфейс воспользуйтесь интерактивным помощником на главном экране:
1.  **DNS-записи**: Зарегистрируйте домен и направьте A-записи домена и субдомена на IP-адрес вашего сервера.
2.  **Привязка домена**: Перейдите в раздел "Добавить домен", укажите имя домена и поддомен для NaiveProxy. Панель выпустит SSL-сертификат и настроит веб-сервер.
3.  **Создание пользователя**: В форме "Новый клиент" укажите имя и привяжите домен.
4.  **Копирование конфига**: Нажмите кнопку "Конфиги" напротив имени клиента, скопируйте VLESS/Hysteria/Naive ссылку или отсканируйте QR-код.

## 🔒 Безопасность и маскировка

Панель использует передовые методы обхода блокировок:
*   Весь трафик маскируется под легитимный HTTPS-трафик домена (decoy).
*   Запросы от посторонних пользователей или сканеров интернет-цензуры перенаправляются на сайт-заглушку (блог, лендинг или портфолио), возвращая код 403/404.
*   Админка защищена случайно сгенерированным секретным префиксом в URL-адресе.
